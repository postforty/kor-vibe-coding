# 네이버 서버 시간 매크로 (초고속)

네이버 서버 시간과 로컬 컴퓨터 시간을 동기화하여, 지정한 목표 시간에 정확히 화면의 특정 좌표들을 초고속으로 클릭하는 파이썬 기반 매크로 프로그램입니다. 주로 수강신청, 티켓팅, 한정판 구매 등 밀리초 단위의 정확성이 요구되는 작업에 활용할 수 있습니다.

## 주요 기능

- **네이버 서버 시간 동기화**: HTTP HEAD 요청과 네트워크 왕복 시간(Ping)을 계산하여 네이버 서버 시간과 내 컴퓨터 시간의 오차를 정밀하게 보정합니다.
- **다중 좌표 설정**: 최대 3개의 클릭할 화면 좌표를 마우스 커서 위치와 `Spacebar` 키 입력을 통해 간편하게 지정할 수 있습니다.
- **초고속 클릭**: `pyautogui.PAUSE = 0`을 적용하여 목표 시간에 도달하는 즉시 지정된 3개의 좌표를 딜레이 없이 순차적으로 클릭합니다.
- **백그라운드 실행 및 최적화**: 멀티스레딩을 적용하여 매크로 대기 중에도 GUI 창이 멈추지 않으며, 남은 시간에 따라 대기 간격을 동적으로 조절하여 CPU 점유율을 최적화합니다.
- **긴급 종료(안전 장치)**: 매크로가 대기 중일 때 `ESC` 키를 누르면 언제든지 즉시 실행을 취소할 수 있습니다.
- **항상 위(Topmost)**: 프로그램 창이 다른 웹 브라우저나 창에 가려지지 않고 항상 최상단에 표시됩니다.

## 요구 사항

- Python 3.12 이상

## 설치 및 실행 방법

1. 프로젝트 디렉토리로 이동합니다.
2. `uv`를 이용하여 프로젝트의 의존성 패키지를 동기화합니다.

```bash
uv sync
```

3. 프로그램을 실행합니다.

```bash
uv run main.py
```

## 사용 방법

1. 프로그램이 실행되면 자동으로 네이버 서버 시간과 동기화를 시도합니다. 동기화가 안 되었거나 오차가 의심되면 **시간 재동기화** 버튼을 클릭하세요.
2. **목표 시간** 입력란에 동작을 원하는 시간을 `HH:MM:SS` (예: 12:00:00) 형식으로 입력합니다.
3. **1~3번 좌표 설정** 버튼을 각각 누른 뒤, 화면에서 클릭할 위치로 마우스를 이동하고 **스페이스바**를 누르면 해당 좌표가 저장됩니다.
4. 모든 설정이 완료되면 **매크로 시작** 버튼을 누릅니다. 상태가 "매크로 대기 중..."으로 변경됩니다.
5. 정확히 목표 시간이 되면 저장된 좌표들을 초고속으로 클릭하고 종료됩니다.
6. 만약 대기 중에 취소하고 싶다면 **ESC** 키를 누릅니다.

## 주의 사항

- 본 프로그램은 교육 및 개인 편의 목적으로 작성되었습니다. 과도한 요청이나 서비스 이용 약관에 위배되는 목적(악의적 트래픽 유발 등)으로 사용하여 발생하는 모든 책임은 사용자 본인에게 있습니다.
- `keyboard` 모듈 특성상 키보드 후킹을 사용하므로, 일부 환경이나 운영체제에서는 관리자 권한 실행이 필요할 수 있습니다.
