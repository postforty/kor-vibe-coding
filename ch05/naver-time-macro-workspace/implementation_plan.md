# 네이버 서버 시간 기반 정밀 매크로 구현 계획

네이버 서버 시간(HTTP Date 헤더)을 기준으로 지정된 시간에 3곳의 좌표를 정밀하게 클릭하는 파이썬(Python) 매크로 프로그램의 구현 계획입니다.

## User Review Required

> [!IMPORTANT]
> 본 계획은 터미널/명령프롬프트(CLI)에서 텍스트 기반으로 동작하는 가벼운 스크립트를 기준으로 작성되었습니다. 만약 버튼이 있는 **창 형태의 프로그램(GUI)** 이 필요하시다면 알려주세요.

## Open Questions

> [!WARNING]
> 다음 사항에 대한 확정이 필요합니다. 답변에 따라 코드가 약간 달라질 수 있습니다.
> 1. **클릭 간격:** 3곳의 좌표를 클릭할 때, 각 클릭 사이에 약간의 지연 시간(Delay, 예: 0.1초)이 필요한가요, 아니면 시스템이 허용하는 한 **최대한 동시에(가장 빠르게)** 연속 클릭해야 하나요?
> 2. **사용자 환경:** 매크로 실행 도중 강제로 중지하고 싶을 때 사용할 '긴급 종료 단축키(예: ESC 키)'가 지정되어 있으면 좋을까요?

## Proposed Changes

프로그램을 역할별로 나누어 유지보수가 쉽게 모듈화하여 구성할 예정입니다.

### Python Packages

필요한 외부 라이브러리 목록입니다.
* `requests`: 네이버 서버에 HTTP 통신을 보내 시간을 가져오기 위해 사용.
* `pyautogui`: 마우스 좌표 인식 및 클릭 이벤트를 발생시키기 위해 사용.
* `keyboard`: (선택) 사용자가 마우스 좌표를 지정할 때 키보드 입력을 감지하기 위해 사용.

### Components & Files

#### [NEW] `requirements.txt`
* 프로젝트 실행에 필요한 패키지 목록을 명시합니다. (`requests`, `pyautogui`, `keyboard`)

#### [NEW] `server_time.py`
* **역할:** 네이버 서버 시간을 확인하고 오차를 계산하는 모듈
* **주요 기능:**
  * `get_naver_server_time()`: `https://www.naver.com`으로 HEAD 요청을 보내 `Date` 헤더 파싱.
  * 네트워크 핑(지연 시간)을 고려하여 오차 범위를 밀리초(ms) 단위로 보정하는 기능.

#### [NEW] `macro_core.py`
* **역할:** 좌표 설정 및 실제 예약 클릭을 수행하는 핵심 모듈
* **주요 기능:**
  * 좌표 지정 도우미: 사용자가 마우스를 원하는 곳에 올리고 특정 키(예: Spacebar)를 누르면 해당 X, Y 좌표를 저장. 이를 3회 반복하여 3곳의 좌표 획득.
  * 초정밀 대기 루프: `(현재 로컬 시간 + 서버 시간 오차)`를 지속적으로 확인하며 목표 시간이 될 때까지 대기 (CPU 점유율을 관리하기 위해 목표 시간 1초 전까지는 `sleep`을 쓰고, 마지막 1초는 `while` 루프로 정밀 대기).
  * 클릭 실행 기능.

#### [NEW] `main.py`
* **역할:** 프로그램 진입점 (동작 흐름 제어)
* **동작 흐름:**
  1. 프로그램 시작 안내 텍스트 출력.
  2. `server_time.py`를 호출하여 현재 네이버 시간과 로컬 시간의 오차 계산.
  3. 사용자에게 목표 시간(예: `10:00:00`) 입력받기.
  4. `macro_core.py`를 호출하여 3개의 클릭 좌표 설정 진행.
  5. 모든 준비가 완료되면 대기 모드 돌입 및 목표 시간에 3개 좌표 연속 클릭 수행.
  6. 결과 출력 후 프로그램 종료.

## Verification Plan

### Manual Verification
1. `requirements.txt`를 통해 패키지 설치를 진행합니다.
2. `main.py`를 실행합니다.
3. 임의의 테스트 시간(현재 시간으로부터 1~2분 뒤)을 설정합니다.
4. 바탕화면의 임의의 위치 3곳의 좌표를 설정합니다.
5. 지정한 시간이 되었을 때, 해당 3곳으로 마우스가 이동하여 클릭이 정확히 발생하는지 확인합니다.
6. 네이버 시계 웹사이트 등을 띄워두고, 매크로의 동작 시간이 네이버 시계와 정확히 일치하는지 눈으로 교차 검증합니다.
