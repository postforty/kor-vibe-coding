# 네이버 매크로 프로그램 구현 완료

요청하신 모든 기능이 반영된 파이썬(Python) 기반의 네이버 서버 시간 매크로 프로그램 개발이 완료되었습니다.

## 구현된 주요 기능
* **초정밀 시간 동기화:** `requests.head()`를 이용해 네이버 서버 시간을 가져오며, 네트워크 왕복 시간(Ping)을 계산하여 밀리초(ms) 단위의 오차까지 보정합니다.
* **직관적인 GUI 환경:** Tkinter를 활용하여 현재 네이버 시간의 실시간 확인, 목표 시간 입력, 3곳의 좌표 설정, 매크로 동작 상태 등을 한눈에 볼 수 있는 쾌적한 창 모드 인터페이스를 제공합니다.
* **초고속 딜레이리스(Delay-less) 클릭:** 시스템이 허용하는 한 가장 빠르게 클릭하기 위해 `pyautogui.PAUSE = 0` 설정을 적용했으며, 1초 미만 남았을 때는 Sleep 없이 루프를 최대한 돌려 0.01초의 틈도 놓치지 않게 설계했습니다.
* **긴급 종료 (ESC):** `keyboard` 백그라운드 훅을 이용하여 대기 중이거나 클릭이 진행 중일 때 언제든 **`ESC` 키**를 누르면 프로그램 동작이 즉시 강제 취소됩니다.

## 실행 가이드

VSCode 터미널이나 명령 프롬프트(CMD)를 열고 다음 명령어를 순서대로 실행해 주세요.

### 1. 프로그램 폴더로 이동
```powershell
cd c:\Users\dandycode\Documents\GitHub\kor-vibe-coding\ch05\naver_macro
```

### 2. 가상환경 생성 및 패키지 설치 (uv 사용)
`uv`를 사용하여 가상환경을 만들고 `requirements.txt`에 명시된 필수 패키지를 설치합니다.
```powershell
uv venv
uv pip install -r requirements.txt
```

### 3. 프로그램 켜기
```powershell
uv run main.py
```

> [!TIP]
> **사용 방법 요약:**
> 1. 프로그램이 실행되면 상단의 파란색 **현재 네이버 서버 시간**이 실시간으로 흘러가는지 확인합니다.
> 2. `1번 좌표 설정` 버튼을 클릭합니다.
> 3. 원하는 위치에 마우스 커서를 두고 키보드의 **Spacebar(스페이스바)**를 누르면 해당 좌표가 기록됩니다. (2번, 3번 좌표도 동일하게 설정)
> 4. `목표 시간`을 입력합니다. (형식: `HH:MM:SS`)
> 5. `매크로 시작` 버튼을 누르면 상태가 **대기 중**으로 변하며, 목표 시간이 되는 순간 3개의 좌표가 딜레이 없이 연속 클릭됩니다. 
> 6. *취소를 원하시면 언제든 `ESC` 키를 누르시면 됩니다.*
