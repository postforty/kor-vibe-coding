# Marp(마프)로 슬라이드 만들기 🚀
### 마크다운으로 빠르고 예쁜 프레젠테이션 제작하기

---

## 1. Marp란?
- **마크다운(Markdown)**으로 작성한 텍스트를 슬라이드로 변환해 주는 도구입니다.
- **장점**: 
  - 디자인에 신경 쓰지 않고 내용에 집중할 수 있습니다.
  - 텍스트 파일 하나로 관리되므로 Git 등 버전 관리에 용이합니다.
  - PDF, HTML, PPTX 등 다양한 포맷으로 내보낼 수 있습니다.
    - ⚠️ **주의**: PPTX로 변환 시 슬라이드가 통째로 '이미지'로 렌더링되어 삽입되므로, 파워포인트 내에서 개별 텍스트나 도형을 직접 수정할 수 없습니다. 모든 내용은 원본 마크다운(`.md`) 파일에서 수정해야 합니다.

---

## 2. 기본 설정 (Front-matter)
문서 맨 위에 `---` 로 감싼 YAML 블록을 추가하여 Marp를 활성화합니다.

```yaml
---
marp: true
theme: default
paginate: true
---
```

- `theme`: default, gaia, uncover 등 기본 테마를 제공합니다.
- `paginate`: 슬라이드 하단에 페이지 번호를 표시합니다.

---

## 3. 슬라이드 나누기
`---` (하이픈 3개)를 입력하면 새로운 슬라이드 페이지로 나뉩니다.

```markdown
# 첫 번째 슬라이드
내용

---

# 두 번째 슬라이드
내용
```

---

## 4. 텍스트 꾸미기
기존 마크다운 문법을 100% 동일하게 사용합니다.

- `#`, `##`, `###` 로 제목 크기를 조절합니다.
- `**굵게**`, `*기울임*`, `~~취소선~~`
- `-` 나 `1.` 을 사용해 글머리 기호(리스트)를 만듭니다.
- `> ` 를 사용해 인용구를 작성합니다.

---

## 5. 슬라이드 레이아웃 조절하기 (Directive)
슬라이드 모양을 제어하는 주석 형태의 명령어를 **지시어(Directive)**라고 합니다.

- `<!-- _class: lead -->`: 현재 슬라이드의 내용을 중앙 정렬하여 타이틀 화면처럼 만듭니다. (`_`가 붙으면 현재 슬라이드에만 적용)
- `<!-- fit -->`: 텍스트가 화면에 꽉 차게 크기를 자동 조절합니다. (예: `# <!-- fit --> 아주 큰 텍스트`)
- `<!-- _style: font-size: 0.8em; -->`: 해당 슬라이드만 폰트 크기나 CSS 스타일을 임의로 조절합니다.

---

## 6. 이미지 삽입과 배경 설정
Marp의 가장 강력한 기능 중 하나인 배경 이미지 제어입니다.

- 기본 이미지 삽입: `![width:400px](이미지주소)`
- 전체 배경으로 지정: `![bg](이미지주소)`
- 화면 분할 (오른쪽에 40% 넓이로): `![bg right:40%](이미지주소)`
- 여러 이미지를 병렬로 배치: `![bg](이미지1) ![bg](이미지2)`

---

## 7. 화면에 맞게 출력 (Export)
- **VS Code Extension**: VS Code에서 `Marp for VS Code` 확장을 설치하면, 우측 상단의 Marp 아이콘 ▷ 모양을 눌러 PDF, HTML, PPTX 등으로 즉시 내보낼 수 있습니다.
- **Marp CLI**: 터미널에서 명령어로 자동화하여 변환할 수도 있습니다.
  `npx @marp-team/marp-cli slide.md --pdf`
